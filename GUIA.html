🚴 Guía de Inicio Rápido - Descarrilados Bike

✅ ¡Felicidades! Tu tienda está lista. Sigue estos pasos para personalizarla.

1️⃣ Cambiar el número de WhatsApp

Este es el paso más importante.

  • Abre el archivo js/script.js
  • Busca esta línea: const whatsappPhone = "51906414352";
  • Reemplaza 51906414352 con tu número de WhatsApp
  • Incluye el código de país (ejemplo: 51 para Perú)
// ANTES: const whatsappPhone = "51906414352"; // DESPUÉS: const whatsappPhone = "51906414352"; // Reemplaza con tu número

2️⃣ Personalizar el nombre de tu tienda

En el archivo index.html, busca y reemplaza:

  • <h1>🚴 Descarrilados Bike</h1>
  • Título de la página: <title>
  • En el footer, la información de contacto

3️⃣ Cambiar los colores

En el archivo css/style.css, busca `:root {` y modifica:

:root { --primary-color: #EF4D48; /* Cambiar este rojo */ --secondary-color: #D90700; /* Cambiar este rojo oscuro */ --dark-color: #2B161B; /* Cambiar este marrón */ --light-gray: #F7F3F5; /* Cambiar fondo */ }

Puedes usar esta herramienta para elegir colores.

4️⃣ Agregar tus productos

En js/script.js, modifica el array products:

const products = [ { id: 1, name: "Tu Producto", price: 500, category: "mtb", // mtb, ruta, accesorios description: "Descripción", emoji: "🚴" // O: 🚵 🛞 🪖 🔧 } ];

Categorías disponibles: mtb, ruta, accesorios

5️⃣ Agregar imágenes de productos (opcional)

  1. Coloca tus imágenes en la carpeta images/
  2. En js/script.js, modifica donde dice:
    <div class="product-image"> ${product.emoji} </div>
  3. Cámbialo por:
    <div class="product-image"> <img src="images/producto${product.id}.jpg" alt="${product.name}"> </div>

6️⃣ Información de contacto

En index.html, busca el footer y actualiza:

  • Número de teléfono
  • Email
  • Enlaces de redes sociales
  • Dirección (opcional)
⚠️ Importante:
  • Siempre guarda los cambios después de editar
  • Recarga la página en el navegador (Ctrl + F5)
  • Prueba el carrito y WhatsApp antes de publicar

7️⃣ Probar tu tienda

  1. Abre index.html en tu navegador
  2. Haz clic en "Comprar Ahora"
  3. Agrega productos al carrito
  4. Haz clic en "Comprar por WhatsApp"
  5. Verifica que tu número y productos aparezcan

🚀 ¿Cómo publicar tu tienda?

Tienes varias opciones:

  • Netlify (Gratis): netlify.com - Arrastra tu carpeta
  • GitHub Pages (Gratis): pages.github.com
  • Vercel (Gratis): vercel.com
  • Hosting propio: Contrata un hosting y sube los archivos vía FTP

📞 Soporte

Si necesitas ayuda:

  • Verifica que todos los números estén correctos
  • Asegúrate de guardar los archivos
  • Recarga la página (Ctrl + F5 o Cmd + Shift + R)
  • Abre la consola del navegador (F12) para ver errores
